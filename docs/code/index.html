Math-Magick-lpp Code¶

Animation¶

OpenSCAD supportsanimating shapes. I decided to generate an animated GIF file using this feature. It appears on the project website. To get this running, click on design -> Animate and set in ta FPS value of 30, and a **Step value of 10. The anitation should start up. There are controls that let you generate animation images that can be combined to build the GIF file.
