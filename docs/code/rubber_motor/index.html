MMlpp Rubber Motor¶

We do not need an accurate model of the motor we will use for this design. Therefore, we will simply model it as a cylinder running between with propeller shaft hook, and the rear hook. The chosen density of this cylinder will make the motor weight about what it should. That will be used in center of gravity calculations. Visually, it might look a bit silly, but YMMV!
