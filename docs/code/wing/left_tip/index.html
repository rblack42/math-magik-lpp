MMlpp Wing Left TipÂ¶

The tips are a bit complex. I decided to use a double-tapered leading and trailing edge to thin our the tips. The circular arc section and the tip rib are all thinner square stock. The circular arc obviously needs to be formed using a template. We will generate that in a later section of these notes.

Herr are the required parts needed for the tip:

  • MMlpp Wing Left Tip Leading Edge
  • MMlpp WIng Left Tip Trailing Edge
  • MMlpp Fuselage
  • MMlpp Wing Left Tip Arc

The positioning data is given in this file:

 1//########################################
 2// arc_pos.scad
 3// (c) 2021 - Roie R. Black
 4//****************************************
 5include <../wing_data.scad>
 6
 7le_pos = [0,0,0,0,0,-90];
 8te_pos = [max_wing_chord - spar_size,0,0,0,0,-90];
 9arc_pos = [tip_radius, -tip_span + tip_radius, 0,0,0,0];
10rib1_pos = [tip_rib_chord/2 + tip_radius,-tip_span,0,0,0,90];
11left_tip_cover_pos = [0,0,0,0,0,0];
